## Common Commands
```
# Download code
$ git clone https://github.com/YueZhao2019/covid-information-system-frontend.git
cd covid-information-system-frontend

# Installation dependency
$ npm install

# Development and debugging
$ npm run dev
Open the browser: http://127.0.0.1:8080/

# Build
$ npm run build
```
## Project structure

```
|-- public                           // Website public directory
|-- src                              // Source directory
|   |-- api                          // Interface management directory
|   |-- assets                       // Public resource directory
|   |-- components                   // Common component directory
|   |-- router                       // Routing configuration directory
|   |-- store                        // State management directory
|   |-- utils                        // JS Tool catalog
|   |-- views                        // View file
|   |-- app.vue                      // Page entry
|   |-- main.js                      // Project entry
```
